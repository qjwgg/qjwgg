#webpy 中自定义 404 页面

几乎每一个网站都有也应该有一个 404 页面，也有一些网站的 404 页面做成了寻找遗失儿童的公益性质的页面。

在 webpy 中自定义 404 页面需要如下步骤：

* url 分发
```
    urls = (
        '...', '...',
        
        '(.+)', 'other',
    )
```
* 404 模板
    * 一个特定的模板，自定义的内容即在此模板内实现。

此 blog 的 404 页面使用的是腾讯提供的公益 404 页面，[点击](http://www.qjwgg.com/webpy/404)查看效果.
